# MAPbIBr
POSCAR files and python scripts for SQS used in our paper, link to be added upon publication.

To use the scripts, ASE and icet for python are required. The scripts also begin with a small unit cell, provided in each directory as `unit.POSCAR`. The resulting SQS produced by the script is `sqs.POSCAR`. It must be pointed out that because icet uses simulated annealing, the outcome of this is not exactly deterministic. We also relaxed each SQS, resulting in `relaxed.sqs.POSCAR`.
